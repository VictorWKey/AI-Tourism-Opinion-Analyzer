AI Tourism Opinion Analyzer

Aplicación de escritorio que utiliza IA y PLN para analizar opiniones turísticas — extrayendo sentimientos, temas, categorías y generando resúmenes inteligentes.

Descargar para Windows Ver en GitHub

Características Principales

Análisis de Sentimientos

Clasificación de sentimientos multilingüe basada en BERT que detecta opiniones positivas, negativas y neutrales en múltiples idiomas.

Modelado de Temas

Descubre temas ocultos en tus reseñas usando análisis jerárquico de temas con BERTopic mejorado con LLMs.

Clasificación por Categorías

Etiquetado multi-categoría de turismo (alojamiento, comida, atracciones, etc.) con modelos BERT afinados.

Resúmenes Inteligentes

Resúmenes generados con LangChain + LLM que sintetizan información clave de cientos o miles de reseñas.

Visualizaciones Interactivas

Gráficos, dashboards, nubes de palabras y reportes exportables — todo lo que necesitas para presentar tus hallazgos.

Privacidad Primero

Se ejecuta completamente en tu máquina con Ollama LLM local. Tus datos nunca salen de tu computadora (a menos que optes por API en la nube).

Comenzar

# Clonar e instalar
git clone https://github.com/victorwkey/AI-Tourism-Opinion-Analyzer.git
cd AI-Tourism-Opinion-Analyzer
npm install

# Ejecutar en desarrollo
npm run start

# Construir instalador de Windows
npm run make:win

La primera ejecución lanza un asistente de configuración que instala Python, modelos de ML y opcionalmente Ollama de forma automática.
