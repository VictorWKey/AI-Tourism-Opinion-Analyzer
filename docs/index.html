Getting Started

# Clone and install
git clone https://github.com/victorwkey/AI-Tourism-Opinion-Analyzer.git
cd AI-Tourism-Opinion-Analyzer
npm install

# Run in development
npm run start

# Build Windows installer
npm run make:win

The first run launches a setup wizard that installs Python, ML models, and optionally Ollama automatically.
